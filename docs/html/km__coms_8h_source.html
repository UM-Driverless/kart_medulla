28// Estructura mensaje, | es solo para visualizar los disintintos campos, no esta
29// en el mensaje
30// | SOF | LEN | TYPE | PAYLOAD | CRC |
31
32// Donde:
33
34// SOF → 1 byte (ej: 0xAA)
35
36// LEN → 1 byte (longitud del payload)
37
38// TYPE → 1 byte (tipo de mensaje)
39
40// PAYLOAD → N bytes
41
42// CRC → 1 byte (checksum simple XOR)
43
44// -------------------------- Tipos de mensajes --------------------------------
45// ## Orin <-> ESP32 Messaging (UTF-8)
46
47// All inter-computer messages are UTF-8 text. Payloads are defined below; line framing and exact
48// field ordering should follow the agreed message definitions when implemented.
49
50// **ESP32 -> Orin (telemetry):**
51// - `actual_speed` (m/s, float)
52// - `actual_acc` (m/s^2, float, signed)
53// - `actual_braking` (0-1, float; interpreted as brake pedal or hydraulic pressure)
54// - `actual_steering` (rad, float)
55// - `mission` (enum; state machine owner TBD)
56// - `machine_state` (enum; mission sub-state)
57// - `actual_shutdown` (0/1, end of SDC loop state)
58// - `esp32_heartbeat`
59
60// **Orin -> ESP32 (commands):**
61// - `target_throttle` (0-1, float)
62// - `target_braking` (0-1, float)
63// - `target_steering` (-1 to 1, float)
64// - `mission` (enum; state machine owner TBD)
65// - `machine_state` (enum; mission sub-state)
66// - `orin_heartbeat`
