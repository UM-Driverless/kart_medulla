1/******************************************************************************
2 * @file Km_rtos.h
3 * @brief Public API for the KM_RTOS task management library.
4 *
5 * This header provides the public interface for the KM_RTOS library, which
6 * offers a structured and safe way to manage FreeRTOS tasks. Through this API,
7 * users can create, delete, suspend, resume, restart tasks, and change task
8 * priorities without directly manipulating internal data structures.
9 *
10 * The library ensures:
11 * - Controlled access to a static internal task array.
12 * - Consistent task creation with default parameters if needed.
13 * - Safe periodic execution of tasks via the KM_RTOS_TaskWrapper.
14 * - Simplified FreeRTOS task management while maintaining thread safety.
15 *
16 * Usage:
17 * 1. Call KM_RTOS_Init() before creating any tasks.
18 * 2. Use KM_RTOS_CreateTask / KM_RTOS_AddTask to define and add tasks.
19 * 3. Manage tasks using KM_RTOS_DeleteTask, KM_RTOS_SuspendTask, KM_RTOS_ResumeTask,
20 * KM_RTOS_RestartTask, and KM_RTOS_ChangePriority.
21 *
22 * Author: Adrian Navarredonda Arizaleta
23 * Date: 24-01-2026
24 * Version: 1.0
25 *****************************************************************************/
